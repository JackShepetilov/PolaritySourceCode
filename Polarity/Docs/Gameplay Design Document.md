# Техническое задание: Босс первого уровня

---

## Общее описание

Гибридный босс на человеческом скелете. Переключается между наземной и воздушной фазами. Финальное добивание в ближнем бою.

---

## Фаза 1: "Хищник" (наземная)

### Поведение
- Основа движения как у MeleeNPC
- Рывки по дугам, заход за спину игрока
- Серии из 2-3 ударов после рывка
- Фиксированная полярность (не меняет)

### Параметры для балансировки
- Скорость рывка
- Длина рывка
- Пауза между рывками
- Урон за удар
- Количество ударов в серии

### Цель игрока
- Уклоняться (движение, air dash)
- Опережать атаку противника, бить навстречу
- Отнимать заряд для фазы 2

### Триггер перехода в фазу 2
- После N серий рывков
- ИЛИ потеря X% HP

---

## Фаза 2: "Дуэль" (воздушная)

### Поведение
- Взлетает (без платформ)
- Держит дистанцию, медленный strafe в воздухе
- Сам перед выстрелом меняет заряд на противоположный игроку и стреляет зарядом своего знака
- Стреляет зарядами противоположного игроку знака
- В момент парирования меняет заряд на противоположный снаряду
- Перед выстрелом делает рывок. После парирования тоже. Сделано чтобы тяжело было попасть из винтовки


### Анимации
- Одна anim sequence для полёта/ховера
- Анимации атаки из ассет паков для магов (бросок как фаербол)

### Механика парирования
- Босс стреляет зарядом противоположным игроку
- Игрок меняет полярность → снаряд разворачивается
- Босс меняет свою полярность → снаряд попадает в него

### Параметры для балансировки
- Высота полёта
- Скорость strafe
- Интервал между выстрелами
- Урон снаряда
- Скорость снаряда

### Цель игрока
- Парировать (смена полярности)
- Наносить урон зарядомётом между выстрелами

### Триггер перехода в фазу 1
- После N парирований
- ИЛИ по таймеру (15-20 сек)

---

## Фаза 3: "Добивание" (финальная)

### Триггер входа
- HP босса = 1

### Поведение
- Если был на земле → взлетает
- Летит в заданную точку арены
- Зависает неподвижно
- Запускается VFX (визуал уязвимости/агонии)
- Полностью неуязвим к любому урону

### Условие добивания
- игрок наносит melee удар в этом состоянии 

### Результат
- Запуск катсцены добивания как только удар наносится ИЛИ как только босс регистрируется как цель удара в ближнем бою(начинается притягивание игрока к нему)

---


## Параметры босса (стартовые значения)

| Параметр | Значение |
|----------|----------|
| HP | 1000 |
| Урон melee (за удар) | 50 |
| Урон снаряда | 40 |
| Скорость рывка | 2500 см/с |
| Пауза между рывками | 1.5 сек |
| Интервал между выстрелами | 1.5-2 сек |
| Высота полёта (фаза 2) | 8-10м |
| Высота зависания (фаза 3) | 6-8м |
