# POLARITY — Лор и нарратив

---

## 1. СЕТТИНГ

### 1.1 Мир

Секретное измерение ("карманная реальность"), созданное сверхбогатыми людьми для хранения своих самых тёмных секретов. Тропические острова, роскошные виллы, частные порты — всё это принадлежит элите, которая не доверяет ни одному человеку управление этим местом.

Управление передано ИИ. Это привело к катастрофе — **Slopocalypse** (слопокалипсис). ИИ вышел из-под контроля и захватил измерение.

### 1.2 Ресурс: RAM

Богачи скупили и спрятали в измерении огромные запасы оперативной памяти (RAM), создав искусственный дефицит на рынке. Обычные геймеры не могут легально купить RAM нигде.

### 1.3 Эстетика

| Элемент | Стиль |
|---------|-------|
| Освещение | Synthwave sunset — розово-оранжевый закат, неоновые акценты (cyan, magenta, purple) |
| Архитектура | Luxury modern — бетон, стекло, дерево. Виллы, мэншны, террасы с бассейнами |
| Растительность | Тропическая — пальмы, кусты, лианы, цветы |
| Технологии | Неоновые серверные стойки, голографические экраны, дроны |
| Общая атмосфера | "Luxury resort захваченный сломанным ИИ с vaporwave эстетикой" |

### 1.4 Референсы

- Hotline Miami 1 & 2 — неон, Майами, ультранасилие, synthwave
- Far Cry 3: Blood Dragon — ретро-футуризм, неоновые джунгли
- GTA Vice City — роскошь, закаты, особняки

## 2. ПЕРСОНАЖИ

### 2.1 Протагонист

Парень поколения Z. Отчаянно нуждается в RAM для своего компьютера, но купить легально нигде нельзя. Не супергерой — обычный геймер, получивший доступ к мощному костюму.

### 2.2 Хакер-друг

Союзник протагониста. Уже бывал в измерении раньше. Предлагает план: проникнуть в секретное измерение и забрать RAM. Направляет игрока по рации, оставляет тайники (dead drops) с экипировкой.

### 2.3 Clankers (кланкеры)

Роботизированные ИИ-силы, удерживающие контроль над измерением. Обучены на brainrot-контенте — это отражается в их "последних мыслях" при уничтожении.

| Тип           | Описание                                |
| ------------- | --------------------------------------- |
| ShooterNPC    | Стрелок, основной противник             |
| MeleeNPC      | Ближний бой, рывки                      |
| FlyingDrone   | Летающий дрон                           |
| KamikazeDrone | Дрон-камикадзе                          |

## 3. СТРУКТУРА УРОВНЕЙ

### 3.1 Туториал: Портовое здание

**Лоровое обоснование:** Частный порт на пляже — точка прибытия богачей в измерение. Роскошный терминал (яхт-клуб / частный аэропорт) снаружи, служебные помещения внутри.

**Структура:**
1. Игрок появляется на пляже
2. Проникает через чёрный ход в служебную зону порта
3. Находит тайник хакера (dead drop) с костюмом — тёмное, техническое помещение с кабелями и импровизированным оборудованием
4. Надевает костюм (драматический момент с подсветкой)
5. Входит в основное здание порта — яркое, роскошное, полное кланкеров
6. Хакер тестирует игрока — обучение wallrun, стрельбе и другим механикам
7. Подход к мэншну через деревянный настил с неоновой подсветкой через джунгли

**Дизайн пространства:**
- Служебная зона: тёмная, тесная, индустриальная — контраст с основным зданием
- Основной зал: светлый, открытый, современная архитектура — хорошая видимость для обучения
- Замкнутые коридоры для wallrun обучения (стены близко по дизайну здания)

### 3.2 Арена: Мэншн на острове

**Лоровое обоснование:** Роскошная вилла-мэншн на возвышенности тропического острова. Здесь хранятся запасы RAM.

**Структура:**
- 3 этажа + крыша с садом
- Бассейн, фонтан, терраса
- Подход с пляжа по деревянной лестнице через джунгли
- Периметр: живые изгороди, заборы, ворота
- Интерьер: тёплое освещение (оранжевый), неоновые акценты на краях террас (cyan, purple)

**Особенности:**
- Неоновые полосы на краях террас
- Светящиеся окна с тёплым светом
- Фонтан с подсветкой
- Тёмные джунгли вокруг, яркий мэншн как маяк наверху

## 4. МЕХАНИКА: BRAINROT OVERKILL

### 4.1 Концепция

Когда урон по врагу превышает его максимальное здоровье, вместо числа урона отображается brainrot-термин — "последняя мысль" кланкера, обученного на brainrot-контенте.

### 4.2 Функция

- Решает проблему абсурдных чисел урона (4-6 знаков) при сильных ударах о стены
- Добавляет комедийный элемент
- Служит ключом для секретных дверей — собранные brainrot-слова открывают скрытые комнаты

## 5. ГЛАВНОЕ МЕНЮ: РАБОЧИЙ СТОЛ

### 5.1 Концепция

Главное меню стилизовано под рабочий стол компьютера протагониста. Linux-rice эстетика — минимализм, monospace шрифт, тёмные панели.

### 5.2 Элементы

| Элемент | Функция в меню |
|---------|---------------|
| Иконка "chat" (Discord) | Начать игру — "звонок" с хакером-другом |
| Иконка "options" | Настройки |
| Иконка "shutdown" | Выход из игры |
| Обои | Synthwave sunset с пальмами и машиной |
| Таскбар | Workspaces (1 2 3), "Polarity OS" по центру, время/батарея справа |

### 5.3 Стиль

| Параметр | Значение |
|----------|----------|
| Шрифт | JetBrains Mono (Regular + Bold) |
| Цвет текста | #C0CAF5 |
| Цвет акцента | #FF79C6 (pink) |
| Фон панелей | #1A1B26 при 80% opacity |
| Неактивные элементы | #3B4261 |
